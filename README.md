# AI Talent Intelligence Platform

An interactive Streamlit dashboard for analyzing Artificial Intelligence job postings across multiple countries.

## Features

- Executive Dashboard
- Geographic Intelligence
- Company Intelligence
- Skills Intelligence
- Salary Analysis
- Hiring Trends
- Job Search
- Interactive Maps
- Export Data

## Dataset

Global AI job postings collected from multiple countries.

## Tech Stack

- Python
- Streamlit
- Plotly
- Pandas
- OpenPyXL

## Installation

```bash
git clone https://github.com/<your_username>/ai-talent-intelligence-dashboard.git

cd ai-talent-intelligence-dashboard

pip install -r requirements.txt

streamlit run app.py
```
